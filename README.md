## About app

PAGE LOADER

Load page from net to upload directory

## Setup

```sh
make install
```

## Run develop mode

```sh
make start-backend
make start-frontend
```
## Run test

```sh
**Run default tests**
make test
```

```sh
**Run test + debug + nock debug**
make test-nock-debug
```

## Run build

```sh
make build
```

```sh
make build
```

## How to use

```sh
**Get help**
page-loader -h
```

```sh
**Get help**
page-loader http://loaded.site -o upload_directory
```


All dependent modules will be download and install in the 'node_modules' folder
